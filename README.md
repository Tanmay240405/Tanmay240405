### 🚀 What I'm working on

- 🎨 **Paint-by-Numbers Research** — an automated PBN generation web app, backed by a Python codebase with a classical pipeline and a SAM+SLIC hybrid pipeline, plus a custom `paintability.py` module for scoring template usability. Working toward a research paper built around formal paintability metrics and a human user study, targeting venues like *Computers & Graphics* and *WACV Applications*.
- 🧸 **Kids Adda** — a web app built with React, Tailwind CSS v4, and react-three-fiber, rendering interactive 3D content in the browser.
- 🧊 **3D Modeling in Blender** — recoloring and reworking imported assets, getting hands-on with material properties and shader nodes.
